# NASA mint dotfiles

My personal Linux dotfiles for Hyprland. Uses default hypr key bindings.

## Includes

- Hyprland
- Waybar
- Wofi
- Kitty
- Fish

## Supported distros

- openSUSE
- Arch Linux
- Fedora

## Install

Clone the repo and run the installer:

```bash
git clone https://github.com/wrgdbtry/dotfiles.git
cd dotfiles
./install.sh
```

What the installer does

    installs needed packages
    creates config directories
    copies configs into ~/.config

Notes

Some packages may have different names depending on distro version.
If something fails, install the missing package manually.
Structure

text

hypr/     -> Hyprland config
waybar/   -> Waybar config
wofi/     -> Wofi config
kitty/    -> Kitty config
fish/     -> Fish config
install.sh
